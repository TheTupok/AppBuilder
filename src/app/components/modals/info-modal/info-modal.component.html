Instructions

App Builder

Application can add, resize, and remove components. You can add components by selecting the desired template and dragging it to the workspace.

Add component

To add an element, click and drag any element from the left area to the workspace

Drag components

To move components in the workplace, you must hold the left mouse button, which will make the element movable

Resize components

At each corner there are elements that make it possible to increase the component

Component Properties

You can change some properties of components. Each element has different properties. You can open the properties by clicking the left mouse button.

Ability to add elements to a container

By creating a container, you can place other objects in it. They will be attached to this container, and when the container is moved, all components inside will also move. Components can be detached from the container by simply pulling it out
